# 🎯 Quiz Application

This is a simple Quiz Application built using Python.

## 🚀 Features
- Multiple choice questions (MCQs)
- User input handling
- Score calculation
- Percentage result display

## 🛠️ Technologies Used
- Python
- Object-Oriented Programming (OOPS)

## ▶️ How to Run
1. Download the project
2. Run the Python file:
   python Project.py
3. Enter answers (A/B/C/D)

## 💡 Future Improvements
- Add timer for each question
- Store scores in database
- AI-based adaptive difficulty
